## Deployment

### Run in local

1. Clone the repository
   ```bash
   git clone https://github.com/kamelliao/myrent.git
   cd myrent
   ```
2. Start backend server
   ```bash
   cd ./backend
   flask run
   ```
3. Start frontend server
   ```bash
   cd ./frontend
   npm run
   ```
4. Open the browser and go to http://localhost:3000
5. Start your journey!
